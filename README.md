TAV-X：Termux 自动化安装脚本🚀 项目简介TAV-X (Termux Automated Venture-X) 是一个为安卓 Termux 环境量身定制的傻瓜式一键安装脚本，旨在简化 SillyTavern 的部署和管理流程。无论您是想在本地使用，还是通过安全隧道跨设备远程访问您的“酒馆”，TAV-X 都能助您一臂之力，让您在安卓手机上随时随地进入您的专属 AI 聊天环境。✨ 项目亮点一键式部署： 自动处理依赖安装、环境配置和项目克隆。TUI 交互式管理界面 & 实时仪表盘： 提供直观的 TUI 菜单，集成了服务状态和实时远程链接显示，一目了然地完成启动、停止、更新、日志查看等所有操作。后台稳定运行（防断线）： 服务使用 setsid nohup 启动，并启用 termux-wake-lock 锁，确保服务在 Termux 后台运行和屏幕熄灭时保持稳定不断线。跨设备访问： 利用 Cloudflare 隧道技术（无需额外配置），实现从任何设备通过安全链接远程进入酒馆。多用户模式： 脚本自动开启多用户功能，您可以添加多个用户，并通过隧道链接分享给您的伙伴。安卓部署优势： 专为 Termux 优化，实现在安卓手机部署，跨设备随时进入酒馆的便捷体验。无损更新： 自动暂存本地修改，确保核心项目更新后本地文件不受影响。⚡ 快速开始您只需要在您的安卓 Termux 终端中执行一条命令，即可完成脚本的下载、授权和运行：curl -LO [https://raw.githubusercontent.com/Future-404/TAV-X/main/st.sh](https://raw.githubusercontent.com/Future-404/TAV-X/main/st.sh) && chmod +x st.sh && ./st.sh
执行上述命令后，脚本将自动完成环境检测和依赖安装，并进入主菜单。🛡️ 安全与多用户设置为了保障您的数据安全和实现跨设备协作，请务必注意以下关键信息：多用户模式已开启： 本脚本已自动在配置文件中开启了多用户（User Accounts）和谨慎登录（Discreet Login）功能。首次登录安全提醒：默认管理员用户名： default-user首次登录无密码： 由于安全考虑，首次运行后 default-user 没有默认密码。立即设置密码： 您必须在登录后，前往管理员设置页面自行设置一个强密码。请务必妥善保管您的密码。分享与协作： 脚本启动远程分享后会生成一个 Cloudflare 隧道链接，您可以将此链接和您创建的用户账号分享给他人，实现多用户同时访问。感谢您使用 TAV-X！在使用过程中如遇到任何问题，欢迎到项目 GitHub 仓库 提交 Issue。
