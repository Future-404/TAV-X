# ♊ GGB2API (TAV-X Stable)

本模块是专为 TAV-X 打造的生产级 Gemini Business 逆向接口。它将 Google 企业版 Gemini 的强大能力转化为完全兼容 OpenAI 标准的 API，并针对移动端 Termux 环境进行了深度性能加固。

---

## ✨ 核心特性

*   🚀 **1.0.0-Stable 版**：经过深度代码审计，移除冗余依赖，核心逻辑轻量且高效。
*   🧠 **深度思考隔离**：完美支持 `Gemini 2.0 Flash Thinking` 等模型。自动识别 `thought` 字段，流式输出支持 `reasoning_content`（兼容 o1 格式），非流式自动包裹 `<think>` 标签。
*   🎨 **完美多模态渲染**：
    *   支持 `gemini-imagen` 原生生图。
    *   支持 `gemini-veo` 原生视频生成。
    *   **自动解码**：内置异步捕获引擎，自动将 Google 私有资源下载并转换为 Base64 嵌入 Markdown，无需配置任何外部存储。
*   🔄 **智能账号调度池**：
    *   **阶梯冷却**：遇到 `429` 自动短冷却并增加惩罚时间，遇到 `401/403` 触发长冷却。
    *   **无感热重载**：采用 mtime 指纹技术，通过 [📥 导入账号] 更新 Cookie 后，服务端无需重启，下一条请求自动加载新账号。
*   🩺 **内置环境诊断**：提供一键体检功能，快速定位依赖缺失、端口占用或虚拟环境故障。

## 🛠️ 快速开始

### 1. 部署与环境
1. 在 TAV-X 菜单中执行 **[安装/更新]**。
2. 模块会自动配置 Python 3.13 虚拟环境。
3. **Termux 用户注意**：若涉及 Rust 编译，系统已自动配置 `ANDROID_API_LEVEL=24` 补丁，请耐心等待 3-5 分钟。

### 2. 账号录入
本模块推荐使用浏览器插件进行 **一键导入**：

*   **PC 端**：使用 Chrome/Edge 并安装 **[Cookie Tool]** 插件。
*   **安卓端**：推荐使用 **[狐猴浏览器 (Lemur Browser)]** 安装上述 Chrome 插件。

**操作步骤**：
1. 登录 Gemini Business 对话界面。
2. 点击 Cookie Tool 插件，选择 **[Copy Cookie (JSON)]**。
3. 在 TAV-X 菜单中执行 **[📥 导入账号]**，按提示输入后直接粘贴并回车。
> **提示**：支持导入无限个账号，系统会自动进行负载均衡。

### 3. 服务自检
启动前建议执行 **[🩺 环境诊断]**，确保所有生产组件显示为 `OK`。

## 💬 客户端接入

*   **Base URL**: `http://127.0.0.1:7860/v1`
*   **API Key**: `sk-business-key` (可在修改配置中自定义)

### 模型列表 (部分)
| 模型名称 | 特性 |
| :--- | :--- |
| `gemini-auto` | 默认模型，自动调度 |
| `gemini-2.5-pro` | 强大的长文本与逻辑推理 |
| `gemini-3.1-pro-preview` | 最新的 3.1 预览版 |
| `gemini-imagen` | **专用生图模型** (直接发送提示词即可) |
| `gemini-veo` | **专用视频生成** |
